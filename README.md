# Construction Clock 🏗️ — v0.2

Операционная система строительной компании. Чистая переписка Check Time по карте 27 блоков.

## Что внутри

- **База**: Supabase-проект `Construction Clock 🏗️` (gzjfjszfdnmaazursppx), 30 таблиц, 7 миграций, RLS по ролям, append-only журнал событий.
- **Edge-функции**: `pin-login` (вход по PIN, анти-перебор), `create-worker` (создание работника менеджером).
- **Приложение (эта папка)**: React + Vite + TypeScript, PWA, мобильный-первый, RU/EN/ES, тёмная тема.

## Экраны v0.1

- Вход: работник по PIN (клавиатура), офис по email
- Дашборд (менеджер): кто на объекте, часы сегодня, активность
- Отметка: выбор проекта → ПРИШЁЛ/УШЁЛ/перерыв, GPS-захват (нет GPS — отметка проходит)
- Проекты + задачи (🔨 работа / 📦 материалы / 🚚 доставка)
- Команда: список, часы, добавление работника с PIN
- Мои часы: сегодня/неделя

## Запуск локально

```bash
npm install
npm run dev        # http://localhost:5173
```

## Продакшн

```bash
npm run build      # dist/ — готовая статика
```

Разместить `dist/` можно на Vercel / Netlify / Cloudflare Pages (бесплатно): подключить репозиторий или перетащить папку. После размещения приложение устанавливается на телефон как PWA.

## Доступы

Учётные данные (логины, PIN-коды, пароли) в репозитории не хранятся — их выдаёт владелец организации. Первый вход и смену пароля см. у владельца.

## Архитектура

Ключи в `src/lib/supabase.ts` — публичный publishable key (безопасен для клиента, доступ ограничен RLS).
Правила системы — в ДНК-документе проекта Claude («dna_construction_clock.md»).
